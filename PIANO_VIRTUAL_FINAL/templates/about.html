Acerca del Piano Virtual Invisible

¿Qué es?

El Piano Virtual Invisible es una aplicación interactiva que utiliza visión por computadora para detectar gestos de las manos y convertirlos en notas musicales, sin necesidad de un piano físico.

Tecnologías utilizadas

  • Python - Lenguaje principal
  • Flask - Framework web
  • WebSockets - Comunicación en tiempo real
  • MediaPipe - Detección de puntos de referencia de las manos
  • TensorFlow - Modelos de aprendizaje automático
  • OpenCV - Procesamiento de imágenes
  • Pygame - Reproducción de audio

Cómo funciona

  1. La cámara captura el video en tiempo real
  2. MediaPipe detecta los puntos de referencia de la mano
  3. El algoritmo determina si el dedo índice está doblado para tocar una nota
  4. La posición del dedo determina qué nota se toca
  5. El sonido correspondiente se reproduce

Características

  • Rango de 5 octavas (DO2 - SI6)
  • Navegación entre octavas mediante gestos
  • Interfaz visual del teclado
  • Detección de manos en tiempo real
  • Reproduce sonidos de piano reales
